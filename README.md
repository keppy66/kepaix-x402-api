# KepAIx x402 API

Educational market intelligence API powered by x402 payments on Algorand.

## Overview

KepAIx provides machine-readable educational market intelligence designed for:

* AI agents
* Applications
* Dashboards
* Research tools
* Developers

The API returns a lightweight market regime snapshot generated by the KepAIx Main Brain.

## Public Endpoint

https://kepaix.com/api/market-regime.php

Returns a public educational market snapshot.

## x402 Paid Endpoint

https://kepaix.com/api/x402-market-regime.php

### Configuration

Network: Algorand Mainnet

Asset: USDC ASA 31566704

Price: 0.01 USDC per request

Protocol: x402

### Example Response

```json
{
  "project": "KepAIx",
  "market_mode": "RISK_OFF",
  "market_state": "CAUTION",
  "confidence": 46.1,
  "risk_score": 66.7,
  "summary": "Educational market regime snapshot"
}
```
